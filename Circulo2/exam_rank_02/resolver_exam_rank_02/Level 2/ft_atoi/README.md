## Subject
```
Expected Files: ft_atoi.c
Allowed functions: None
--------------------------------------------------------------------------------

Escribe una función que convierte el argumento de tipo cadena str a un número entero (int) y lo devuelve.
Funciona de manera similar a la función estándar atoi(const char *str), consulta el man.

Tu función debe ser declarada de la siguiente manera:

int ft_atoi(const char *str);

--------------------------------------------------------------------------------
```


## Subject
```
Expected Files: ft_atoi.c
Allowed functions: None
--------------------------------------------------------------------------------

Write a function that converts the string argument str to an integer (type int) and returns it.
It works much like the standard atoi(const char *str) function, see the man.

Your function must be declared as follows:

int ft_atoi(const char *str);

--------------------------------------------------------------------------------
```
