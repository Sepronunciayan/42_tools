## Subject
```
Expected Files: max.c
Allowed functions:
--------------------------------------------------------------------------------
Escribe la siguiente función:

int max(int* tab, unsigned int len);

El primer parámetro es un arreglo de enteros, el segundo es el número de elementos en el arreglo.
La función devuelve el número más grande encontrado en el arreglo.
Si el arreglo está vacío, la función devuelve 0.
--------------------------------------------------------------------------------
```
