## Subject
```
Expected Files: maff_revalpha.c
Allowed functions: write
--------------------------------------------------------------------------------
Escribe un programa que muestre el alfabeto en orden inverso, con las letras pares en mayúsculas y
las letras impares en minúsculas, seguido de un salto de línea.

Ejemplo:
$> ./maff_revalpha | cat -e
zYxWvUtSrQpOnMlKjIhGfEdCbA$
--------------------------------------------------------------------------------
```

## Subject
```
Expected Files: maff_revalpha.c
Allowed functions: write
--------------------------------------------------------------------------------
Write a program that displays the alphabet in reverse, with even letters in uppercase,
and odd letters in lowercase, followed by a newline.

Example:
$> ./maff_revalpha | cat -e
zYxWvUtSrQpOnMlKjIhGfEdCbA$
--------------------------------------------------------------------------------
```
