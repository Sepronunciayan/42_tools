## Subject
```
Expected Files: ft_itoa.c
Allowed functions: malloc
--------------------------------------------------------------------------------
Escribe una función que reciba un entero y lo convierta en una cadena de caracteres
terminada en nulo (es decir, terminada en \0).

La función debe devolver el resultado en un array de tipo char que tú mismo debes
reservar dinámicamente.
Debe declararse así:

char *ft_itoa(int nbr);
--------------------------------------------------------------------------------
```

Write a function that takes an int and converts it to a null-terminated string.
