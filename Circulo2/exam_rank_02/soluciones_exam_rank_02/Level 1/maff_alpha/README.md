## Subject
```
Expected Files: maff_alpha.c
Allowed functions: write
--------------------------------------------------------------------------------
Escribe un programa que muestre el alfabeto, con las letras pares en mayúsculas y las
letras impares en minúsculas, seguido de un salto de línea.

Ejemplo:
$> ./maff_alpha | cat -e
aBcDeFgHiJkLmNoPqRsTuVwXyZ$
--------------------------------------------------------------------------------
```


## Subject
```
Expected Files: maff_alpha.c
Allowed functions: write
--------------------------------------------------------------------------------
Write a program that displays the alphabet, with even letters in uppercase, and odd
letters in lowercase, followed by a newline.

Example:
$> ./maff_alpha | cat -e
aBcDeFgHiJkLmNoPqRsTuVwXyZ$
--------------------------------------------------------------------------------
```
