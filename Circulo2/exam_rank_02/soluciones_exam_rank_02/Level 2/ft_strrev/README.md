## Subject
```
Expected Files: ft_strrev.c
Allowed functions:
--------------------------------------------------------------------------------

Escribe una función que invierta (en el lugar) una cadena. Debe devolver su parámetro.
Tu función debe ser declarada de la siguiente manera:

char *ft_strrev(char *str);

--------------------------------------------------------------------------------
```

## Subject

```
Expected Files: ft_strrev.c
Allowed functions:
--------------------------------------------------------------------------------

Write a function that reverses (in-place) a string. It must return its parameter.
Your function must be declared as follows:

char *ft_strrev(char *str);

--------------------------------------------------------------------------------
```
